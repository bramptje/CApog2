# AGENTS

- Keep all images in `img/`.
- Keep all JavaScript in `js/`.
- Always check for SEO opportunities.
- Check for broken links.
- Prefer relative links for internal pages.
- Maintain meta descriptions, canonical URLs, and JSON-LD when touching pages.
- Keep external assets minimal; avoid new JS unless it buys UX.
- Use a local HTTP server for testing (avoid `file://`).
- Keep HTML simple.
- Use native HTML elements; avoid extra CSS classes.
